Justab

This project is a side project of mine. It all started with me wanting to have my calendar on every new tab. After I started this project I got enthousiastic about it and decided to improve and expand it with other services.

JusTab is a Google Chrome extension which turns every new tab into an overview of all my favorite services from the web. The extension has intergration for services like: Google Calendar, Gmail, Facebook and Designernews.

Thanks to this selfmade extension I'm able to be more productive. I don't have to open 9 tabs to view everything but I can just open one. At this moment I have more then 100 weekly users.

For more information about this project and a download go to: the Chrome Webstore .
