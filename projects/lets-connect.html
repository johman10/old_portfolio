Let's Connect

Let's Connect is a platform meant to do bible studies. Normally those studies are done with pencil and paper but Let's Connect digitalizes this.

I build this platform in cooperation with Picoreclame . Picoreclame build the back-end while I did the front-end. This project is still being used to this day.

This project is build in Ruby on Rails. The platform accessible with different roles and each role has it's own features available. It's also responsive so it's easy to use on mobile and tablet.
